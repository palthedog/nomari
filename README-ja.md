# Nomari - GTO 計算機

[English](./README.md) / [日本語](./README-ja.md)

---

Nomariは、ストリートファイター6などの格闘ゲーム向けのゲーム理論計算ツールです。
様々な状況におけるゲーム理論最適（GTO）戦略を計算できます。

Nomariは以下のような質問に答えるのに役立ちます：
- 以下のコンボで起き攻め状況を含めて、どちらがより多くのダメージを期待できるか？
    - 一方のコンボは高いダメージ（例：2000ダメージ）だけど起き攻めはついてこない
    - もう一方のコンボは低いダメージ（例：1500ダメージ）だけど、その後の起き攻め攻撃が可能
        - ❓️完璧にプレイした場合、その後の起き攻め攻撃のダメージの期待値は？
    - ❓️コンボと起き攻め合わせたら、どちらのコンボがより高い期待ダメージを持つか？
- ❓️攻撃側としてマリーザが起き攻め状況で取るべき最適な混合戦略は？
    - 攻撃側の選択肢：
        - ドライブラッシュ > コマンド投げ
        - ドライブラッシュ > 4HPコンボ
        - ドライブラッシュ > ガード
    - 防御側の選択肢：
        - ガード
        - ジャンプ
        - 無敵技（OD 昇竜拳など）
    - ❓️ダメージ/勝率の期待値を最大化するために、「コマンド投げ」をどのくらいの頻度で選択すべきか？
    - GTO防御側の戦略も計算できます！
        - ❓️期待ダメージを最小化するために、「ガード」をどのくらいの頻度で選択すべきか？
