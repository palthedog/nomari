# ゲーム木ができるまで

## ゲームの定義
GameDefinition message を使ってゲーム木を作るのに必要な情報を記述します。
必要なものは
- 初期状態
-- 自キャラ残り体力: 5000
-- 相手キャラ残り体力: 4000
- 起こり得る状況一覧
-- 密着起き攻め+4F
-- ニュートラル(投げ抜け後など)
- 各状況で自分と相手が取りうる行動一覧
-- @ 密着起き攻め+4F
--- 自分
---- 中パン
---- 投げ
--- 相手
---- ガード
---- グラップ
---- 暴れ
- 行動の組み合わせとその結果起こることの定義
-- 中パン v.s. ガード
--- ニュートラルに移行
-- 中パン v.s. 暴れ
--- 相手に2000ダメージ
--- 密着起き攻め+4Fに移行

これらの情報から自分や相手の体力の変化を含めた状況を元にゲーム木全体を構築します。
